# Typography Guide

## Font Pairings

Choose a header font with personality and pair it with a clean body font. Don't default to Arial for everything.

| Header Font | Body Font | Vibe |
|-------------|-----------|------|
| Georgia | Calibri | Classic, literary, trustworthy |
| Arial Black | Arial | Bold, modern, impactful |
| Calibri | Calibri Light | Clean, corporate, minimal |
| Cambria | Calibri | Academic, formal, structured |
| Trebuchet MS | Calibri | Friendly, approachable, modern |
| Impact | Arial | High-energy, marketing, bold |
| Palatino | Garamond | Elegant, editorial, refined |
| Consolas | Calibri | Technical, developer, data-focused |

## Font Sizes

| Element | Size | Weight |
|---------|------|--------|
| Slide title | 36-44pt | Bold |
| Section header | 20-24pt | Bold |
| Body text | 14-16pt | Regular |
| Large stat / KPI | 60-72pt | Bold |
| Captions / footnotes | 10-12pt | Regular, muted color |

## Rules

1. **Size contrast matters** — titles at 36pt+ must visually dominate 14-16pt body text
2. **One header font + one body font** — never mix more than two font families
3. **Left-align body text** — center only titles and standalone stats
4. **Muted captions** — use a lighter shade of the text color for captions
5. **Bold for emphasis** — use bold weight for key terms, not italic or underline
6. **Line spacing** — keep body text line spacing at 1.2-1.5x for readability

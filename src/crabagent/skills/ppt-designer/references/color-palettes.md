## Usage Tips

- **Dark/light sandwich**: Dark background for title + closing slides; light background for content slides. This creates visual rhythm.
- **Premium feel**: Commit to dark throughout for a premium, tech-keynote style.
- **Accent usage**: Use accent color sparingly — for key numbers, important icons, or call-to-action elements only.
- **When passing colors to `office_edit` props**: Use hex WITHOUT the `#` prefix (e.g., `1E2761` not `#1E2761`).
